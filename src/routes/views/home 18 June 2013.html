About...

What...

The Supplier Portal is where your customers can view and submit their Invoices and other documents to your company, so they can get processed by eFLOW.


Who...

Developed by Eduardo Freitas.

The company behind the Supplier Portal is Top Image Systems.

Technology...

Works with eFLOW and eFLOW Invoices.

Developed with open source and proprietary technology:


- eFLOW SDK's - Easy to use and powerful SDKs that extend eFLOW beyond imagination.

- Node - Platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications.

- Express - Sinatra inspired web development framework for node.js.

- jQuery - Fast and concise JavaScript Library that simplifies HTML development.

- .NET Framework - Integral part of many applications running on Windows and provides common functionality for those applications to run.
