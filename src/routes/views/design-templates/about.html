About...

What...

The Supplier Portal is where your customers can submit their Invoices to your company, so they can get processed by eFLOW.


Who...

Developed by Eduardo Freitas. Works with eFLOW Invoices.

The company behind the Supplier Portal is Top Image Systems UK.


Technology...

Developed with open source and proprietary technology:


- Node - Platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications.

- Express - Sinatra inspired web development framework for node.js.

- MongoDB - Scalable, high-performance, open source NoSQL database written in C++.

- Mongoose - Elegant mongodb object modeling for node.js.

- Amazon Web Services - (AWS) delivers a set of services that together form a reliable, scalable, and inexpensive computing platform “in the cloud”.

- jQuery - Fast and concise JavaScript Library that simplifies HTML development.

- Knockout.js - Simplifies dynamic JavaScript UIs by applying the Model-View-View Model (MVVM) pattern.

- .NET Framework - Integral part of many applications running on Windows and provides common functionality for those applications to run.

- eFLOW - The most flexible and advanced ECM / Automated Invoice processing platform in the market.
